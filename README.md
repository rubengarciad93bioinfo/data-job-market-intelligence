# Data Job Market Intelligence Dashboard

Portfolio project focused on analysing the Data / AI job market using SQL, Python and dashboarding.

## Main goal

Understand which skills, tools and role profiles are most demanded in the Data Analyst / Data Scientist job market.

## Planned analysis

- Role classification: Data Analyst, Data Scientist, ML Engineer, Data Engineer, BI Analyst.
- Skill extraction from job descriptions.
- SQL-based market analysis.
- Salary and remote-work analysis where available.
- Final dashboard / portfolio page.

## Tech stack

- Python
- pandas
- DuckDB
- SQL
- Jupyter
- GitHub
- Optional: Streamlit / Lovable
